# DJCYTools 当前进度

更新日期：2026-05-03

## 产品定位

DJCYTools 当前聚焦“一人公司一键生成短视频”：用 DeepSeek 生成结构化短剧剧本，用 Doubao Seedance 2.0 生成真实视频，首页动态展示已生成成片，工作台保留尽量少的核心操作。

## 已完成

- 首页按更简洁的落地页结构更新，首屏改为真实视频轮播。
- 工作台主入口收敛为项目和视频两类核心任务。
- DeepSeek 剧本生成与定向改写已接入服务端代理。
- Doubao Seedance 2.0 真实视频任务已接入火山方舟 `contents/generations/tasks`。
- 真实视频测试阶段固定 15 秒，不使用本地模拟预览冒充结果。
- 成功生成的视频会下载到 `data/generated-videos/`，并通过首页轮播展示。
- 结构化剧本与分镜建议已合并，减少重复内容。
- SQLite 已作为本地主存储，已补充 PostgreSQL 迁移脚本。
- 模板库已扩展到 120 个，覆盖 18 个短剧类型。
- README 已更新为当前产品方向和 120 个模板统计。

## 当前不做

- 团队协作界面
- 第三方交付接口界面
- 版本实验
- 评分和增长模块
- 合规和相似度模块
- 社区模板市场
- 本地模拟视频预览

## 下一步

- 继续压缩工作台非核心信息密度，只突出“选模板 -> 生成剧本 -> 生成真实视频 -> 导出”。
- 增强 Seedance 提示词与当前剧本分镜的一致性，让生成视频更贴合用户输入的剧情。
- 为真实视频任务补充更清晰的失败原因和账号限制指引。
- PostgreSQL 正式上线前做一次真实数据库迁移演练。
